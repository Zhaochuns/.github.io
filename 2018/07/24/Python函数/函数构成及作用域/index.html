函数的构成

## 函数  是一个容器，里面包含了代码执行语句，还有一系列的变量数据

  • def

  • 参数

    • 形参：女嘉宾 (每一位女嘉宾都要和一个男嘉宾配对，顺序一定要对)

    • 实参：男嘉宾 具体给形参赋值的，#上错花轿嫁对郎 运气

    • 必备参数

    • 命名参数

    • 缺省参数

    • 不定长参数

  • 返回值

    • 函数内部的一些操作，不能影响外界的事务
    • return 语句 在函数执行完成之后返回一个结果
    • return 会终止函数运行
    • 函数调用完成之后会返回结果

    • 默认的函数如果没有返回值，那么返回None

    #func()()
    #func: -> 内部定义好的func
    func() #重新改造之后的调用
